# 超低温温度测量模块简介

## 产品概述

Ultra-TM02型超低温温度测量模块是用于**二极管式温度探头**进行超低温测量的变换模块，用于将二极管的节电压转换为对应的温度并进行输出和显示。本模块适用于二极管式温度探头，且允许用户自己更新探头的分度表，适用于多种环境下的低温测量。

### 电路模块说明

| 模块 | 功能描述 |
|------|----------|
| **电源电路** | 将AC 220V转换成+9V、+5V以及+3.3V供模块中的各个电路使用 |
| **主控电路** | 用于控制其它电路协同运行，实现模块的基本功能 |
| **LCD电路** | 主控电路通过USART与LCD电路进行通讯，以显示模块工作状态 |
| **RS485电路** | 主控电路使用USART配合RS485电路实现模块的通讯功能 |
| **USB电路** | 主控电路使用USART配合USB电路实现模块的通讯功能 |
| **隔离电路1** | 主控电路使用SPI经隔离电路1与D/A电路和A/D电路通讯，并控制它们进行输出和输入 |
| **隔离电路2** | 主控电路使用IO经隔离电路2控制(4-20)mA电流源进行电流输出 |
| **D/A电路** | 数模转换，输出电压信号 |
| **A/D电路** | 模数转换，采集输入信号 |
| **V/I电路** | 电压到电流转换 |
| **0.05%电压参考电路** | 提供高精度参考电压，保障模块的指标 |
| **保护电路** | 保护D/A电路、A/D电路、V/I电路、输出电路、输入电路以及0.05%电压参考电路 |

### 信号流程

1. **D/A电路**在主控电路的控制下输出电压信号
2. 该信号经过**V/I电路**变换为电流信号
3. 电流信号经过**输出电路**输出到探头
4. **输入电路**将探头中二极管的节点压信号引入
5. 经**A/D电路**变换为数字量
6. 进行换算与显示

## 2.4 电流源的选择与微调

### 电流源选择

用户可选择**10μA**或**17μA**作为模块输出电流。

| 电流源 | 适用场景 |
|--------|----------|
| 10μA | 标准二极管探头 |
| 17μA | 特定型号二极管探头 |

### 电流源微调

若校准后发现电流存在偏差，可进行微调：

1. 选择对应的电流源（10μA或17μA）
2. 在调整值输入框中输入调整量
3. 配合数字多用表测量实际输出
4. 调整至结果达标后点击**【电流源调整】**
5. 点击**【存储参数】**按钮保存设置至模块Flash

---

## 2.5 4mA~20mA输出调整

### 功能说明

本模块支持**4mA至20mA节点调整**功能：

| 设置项 | 说明 |
|--------|------|
| 4mA代表温度点 | 用户可自定义4mA电流对应的最低温度点 |
| 20mA代表温度点 | 用户可自定义20mA电流对应的最高温度点 |

> 无需使用模块出厂默认的调整方式

### 操作方法

1. 在对应输入框中输入温度点数值
2. 点击**【(4-20)mA调整】**按钮
3. 点击**【存储参数】**按钮完成设置

## 附录C：常见问题解答

### Q1: 如何选择正确的串行口？

A: 连接模块后，在设备管理器中查看新增的COM端口号，然后在软件中选择对应的串行口。

### Q2: 为什么获取不到设备ID？

A: 请检查：
1. USB线缆连接是否正常
2. 驱动是否正确安装
3. 串行口选择是否正确
4. 是否点击了【停止采集】按钮

### Q3: 如何校准电流源？

A: 
1. 选择对应的电流源（10μA或17μA）
2. 使用高精度数字多用表测量实际输出电流
3. 在调整值中输入偏差量
4. 点击【电流源调整】后再次测量
5. 调整满意后点击【存储参数】保存
