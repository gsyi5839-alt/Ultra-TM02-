## 三、接线指南

### 3.1 电源连接

```
┌─────────────┐      ┌─────────────────┐      ┌──────────┐
│ 24V开关电源  │─DC头─→│ DC母座转端子线   │─端子─→│ 主板CN14  │
│  DC24V 2A   │      │ DC5.5×2.1转XH2.54│      │ 24V输入  │
└─────────────┘      └─────────────────┘      └──────────┘
```

**连接步骤**：
1. 将24V电源的DC公头插入DC母座转端子线的母座
2. 将XH2.54-2P端子插入主板CN14接口
3. 上电前确认极性正确（红=+24V，黑=GND）

### 3.2 显示屏连接

```
┌─────────────┐      ┌──────────────┐      ┌─────────────┐
│  3.5寸串口屏 │      │ 1.25转2.54   │      │  主板FPC1   │
│ (2.54mm排针) │←杜邦→│   转接线     │←1.25→│ (1.25mm座)  │
└─────────────┘      └──────────────┘      └─────────────┘
```

**连接步骤**：
1. 将转接线的1.25mm端子插入主板FPC1接口
2. 将转接线的杜邦母头逐一插入串口屏的排针

**引脚对应关系**：

| 转接线颜色 | FPC1引脚 | 信号 | 串口屏引脚 |
|------------|----------|------|------------|
| 线1 | 1 | GND | GND |
| 线2 | 2 | +5V | VCC |
| 线3 | 3 | RX | TX |
| 线4 | 4 | TX | RX |

⚠️ **注意**：TX/RX需要交叉连接（主板TX→屏幕RX，主板RX→屏幕TX）

### 3.3 USB连接

```
电脑USB口 ──Type-C数据线──→ 主板LCM2 (Type-C接口)
```

### 3.4 调试器连接

```
┌──────────────┐                   ┌─────────────┐
│  ST-Link V2  │                   │  主板 H1    │
├──────────────┤    杜邦线         ├─────────────┤
│    SWCLK     │ ←───────────→   │    SWCLK    │
│    SWDIO     │ ←───────────→   │    SWDIO    │
│    GND       │ ←───────────→   │    GND      │
│    3.3V      │ ←───(可选)───→  │    3.3V     │
└──────────────┘                   └─────────────┘
```

### 3.5 测试探头连接（1N4148）

```
┌─────────────────────────────────────────────────────────┐
│                      4线制接法                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   XH2.54-4P端子线          鳄鱼夹          1N4148二极管  │
│   ┌──────────┐           ┌─────┐         ┌─────┐      │
│   │  V+ (电压+) │ ←──红色──┤     ├──红色──→│ 正极 │      │
│   │  I+ (电流+) │ ←──黄色──┤     ├──黄色──→│ 正极 │      │
│   │  I- (电流-) │ ←──绿色──┤     ├──绿色──→│ 负极 │      │
│   │  V- (电压-) │ ←──黑色──┤     ├──黑色──→│ 负极 │      │
│   └──────────┘           └─────┘         └─────┘      │
│                                                         │
└─────────────────────────────────────────────────────────┘

简化说明：
- V+和I+ 接二极管正极（有色带的一端为负极，无色带的为正极）
- V-和I- 接二极管负极
```

## 四、测试步骤

### 4.1 硬件连接测试

1. **检查电源**
   - 连接24V电源到CN14
   - 检查各路电压是否正常（5V、3.3V）

2. **检查显示屏**
   - 连接杜邦线到显示屏和FPC1
   - 上电后屏幕应显示界面

3. **检查USB通讯**
   - 连接Type-C数据线
   - 电脑应识别出虚拟串口

4. **检查调试接口**
   - 连接ST-Link V2
   - 使用Keil或STM32CubeIDE检测芯片

### 4.2 功能测试

1. **温度测量测试**
   - 连接1N4148二极管作为测试探头
   - 观察屏幕显示电压值
   - 室温下应显示约0.6-0.7V

2. **上位机通讯测试**
   - 打开TempDownloader软件
   - 选择正确的串口
   - 点击"获取设备ID"测试通讯

## 五、注意事项

### 5.1 安全注意事项

⚠️ **警告**：
- 24V电源接入前请确认极性正确
- 避免短路，可能损坏主板
- 调试时注意静电防护

### 5.2 连接注意事项

1. **电源极性**：确认24V正负极
2. **串口交叉**：TX接RX，RX接TX
3. **SWD接线**：确认SWCLK、SWDIO对应正确
4. **探头极性**：二极管有极性，注意正负极

### 5.3 测试注意事项

- 1N4148仅用于功能验证，不用于精确测量
- 专业超低温测量需使用DT-670/DT-470等专用探头
- 测试时避免触摸二极管，体温会影响测量

## 六、故障排除

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无法上电 | 电源极性反接 | 检查24V正负极 |
| 屏幕不亮 | 接线错误/电源不足 | 检查杜邦线连接 |
| USB无法识别 | 数据线问题/驱动缺失 | 更换数据线/安装驱动 |
| ST-Link连接失败 | 接线错误/芯片损坏 | 检查SWD接线 |
| 无温度显示 | 探头未连接/极性反 | 检查二极管连接 |
