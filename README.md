# Ultra-TM02 超低温温度测量模块

## 📋 项目简介

Ultra-TM02是一款高精度超低温温度测量模块，专门用于二极管式温度探头的超低温测量，支持用户自定义探头分度表。

## 🎯 主要技术指标

| 参数 | 规格 |
|------|------|
| 温度测量范围 | 1.4K ~ 500K (-271.75℃ ~ 226.85℃) |
| 显示分辨率 | 0.001℃ |
| 最大允许误差 | 低温段 ±1℃，高温段 ±2℃ |
| ADC精度 | 24位 |
| 内部电流源精度 | ≤0.05% |
| 激励电流 | 10μA / 17μA 可选 |
| 输出形式 | 4-20mA（两线制） |
| 供电电压 | 24V DC |
| 显示屏 | 3.5寸LCD串口屏 |

## 🔧 硬件规格

### MCU
- **型号**: STM32F411RET6
- **内核**: ARM Cortex-M4
- **主频**: 100MHz
- **Flash**: 512KB
- **RAM**: 128KB

### 外设
- **SPI1**: ADC/DAC通讯 (12.5MHz)
- **USART6**: LCD串口屏通讯 (115200bps)
- **USB**: CDC虚拟串口
- **GPIO**: 片选、LED、DRDY等

## 💻 软件架构

### 固件分层架构
```
┌─────────────────────────────────────────┐
│              应用层 (App)                │
│  温度测量 | 数据处理 | 参数管理 | 通讯    │
├─────────────────────────────────────────┤
│             服务层 (Service)             │
│   ADC服务 | DAC服务 | LCD服务 | USB服务  │
├─────────────────────────────────────────┤
│              驱动层 (BSP)                │
│  SPI驱动 | UART驱动 | GPIO驱动 | Flash   │
├─────────────────────────────────────────┤
│             硬件层 (HAL)                 │
│           STM32 HAL / LL 库              │
└─────────────────────────────────────────┘
```

### 上位机软件
- **语言**: Python 3.x
- **GUI框架**: PyQt5
- **功能**: 设备连接、参数设置、数据采集、分度表下载

## 🚀 快速开始

### 固件编译
1. 安装 STM32CubeIDE
2. 打开 `Ultra_TM02/Ultra_TM02.ioc`
3. 点击 Project → Build All

### 上位机运行
```bash
cd TempDownloader
pip install -r requirements.txt
python main.py
```
